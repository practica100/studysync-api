# StudySync API

API REST desarrollada con Node.js y Express para gestionar grupos de estudio.

## Tecnologías usadas

- Node.js
- Express.js
- MVC

## Endpoints

GET /api/grupos

GET /api/grupos/:id

POST /api/grupos

PUT /api/grupos/:id

DELETE /api/grupos/:id

## Ejemplo JSON

{
  "nombre": "Grupo Física",
  "materia": "Física II",
  "integrantes": 4
}
